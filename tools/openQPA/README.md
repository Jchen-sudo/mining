- QPA.py:程序入口，双击该文件将运行起程序
- window.py:与js交互，界面将直接调用该文件里面的函数
- dpcap.py:读取包文件，解析报文
- analysis2.py:分析特征
- calc.py:函数库
- var.py:参数数据
